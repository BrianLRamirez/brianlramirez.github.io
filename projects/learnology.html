Learnology: Using data to inform learning

For teachers to be truly helpful in transforming student outcomes, they need to be aware of what each student is thinking and knowing, to construct meaning and meaningful experiences in light of this knowledge, to have enough knowledge of the content to provide feedback such that the student moves progressively through curriculum levels. Technology and learning [using] data visualization can be a key tool in facilitating this. Learnology aims to be this tool.

Learnolgy is a learning dashboard that my team and I built as a class project. Learnology is a learning dashboard that gathers a lot of data on student learning and uses said data to provide actionable feedback for teachers.
