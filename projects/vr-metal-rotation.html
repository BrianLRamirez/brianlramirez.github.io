MIT VR-MRT: An oculus quest VR experience

The VR Mental Rotation test is a project that was born at the MIT STEP Lab where I worked with the Collaborative Learning Environments in Virtual Reality Team to create this research project under the supervision of Dr. Meredith Thompson, PhD.

The VR-MRT is a virtual reality application that aims to reproduce the popular mental rotation test created by Shepard and Metzler's in 1971, but rather than using paper to test spatial abilities, we use virtual reality headsets. This project was built with C# on Unity and it is compatible with most VR headsets and it has been optimized for the Oculus Quest. The application also has extensive data logging which we hope will help researchers capture rich data.
